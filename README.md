## 🛠️ Herramientas Incluidas

### 📋 ECE-DES (1.9MB)
**Expediente Clínico Electrónico para Desastres**

Sistema completo de registro y triage para saldos masivos de víctimas:
- Registro rápido de pacientes
- Sistema de triage START
- Base de datos SQLite integrada
- Exportación a Excel
- Trazabilidad clínica completa

### 📊 Dashboard Analítico (952KB)
**Tablero de Control para Puesto de Mando**

Panel analítico para toma de decisiones:
- Métricas en tiempo real
- Gráficas de ocupación
- Reportes ejecutivos
- Análisis de trazabilidad

### 🏷️ Motor de Impresión (13KB)
**Generación de Tarjetas START**

Crea tarjetas físicas de triage:
- Plantilla A4 estándar
- 4 tarjetas por hoja
- Impresión en cascada
- Útil sin electricidad

### 🔧 Generador SCI-H (48KB)
**Tarjetas de Acción para Roles**

Genera tarjetas para el Sistema de Comando de Incidentes Hospitalario:
- Comandante
- Operaciones
- Logística
- Planificación

### 📖 Guía Operativa (41KB)
**Manual del Hospital de Nunca Jamás**

Guía interactiva con protocolos operativos:
- Procedimientos estandarizados
- Referencias rápidas
- Soporte SCI-H

### 🎯 Simulador (65KB)
**Plataforma de Evaluación de Simulacros**

Sistema de entrenamiento y evaluación:
- 3 escenarios de emergencia
- Evaluación en tiempo real
- Reportes de desempeño
- Retroalimentación automática
