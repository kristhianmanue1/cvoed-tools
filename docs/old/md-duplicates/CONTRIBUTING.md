# Contributing to CVOED-Tools

## Desarrollo

1. Editar código en src/
2. Ejecutar `./build.sh`
3. Probar HTMLs en dist/
4. Si funciona, hacer commit

## Restricciones Críticas

- NO romper portabilidad (doble clic debe funcionar)
- NO agregar dependencias externas
- NO usar ES6 modules en código final
- Mantener 100% offline

Ver docs/DEVELOPMENT.md para guía completa.
